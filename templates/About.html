Know More About Us

The Vision Security solution is a tool for student housing that enables Secure entry to rooms using facial recognition and QR code scanning. •We have created a programme for the Raspberry Pi that enables the motion sensor to activate the camera for QR code scanning and ultrasonic range detection for facial recognition. •With their student ID and a photo, students must register on the app before having their information entered into a database. The programme creates a QR code and emails it to the registered email address.
